# **App Name**: VideoVerse Portfolio

## Core Features:

- Video Upload: Allow users to upload their videos (Rills and full-length videos)
- Video Display: Display videos in an organized and visually appealing format.
- Video Categorization: Allow users to categorize videos into Reels and Full Videos sections.
- Interactive tool description Generator: AI powered feature that generates titles, short descriptions, and suggests tags for videos. Tool should consider the video's content, style, and overall theme to ensure relevant and effective metadata.
- Contact form: Allow potential clients to reach out for project offers.

## Style Guidelines:

- Primary color: Deep blue (#2E3192) to convey professionalism and trust. This choice references classic corporate color schemes in art and design.
- Background color: Light gray (#EEEEEE) for a clean and modern backdrop.
- Accent color: Soft purple (#A64AC9) to highlight key elements and call to actions.
- Body and headline font: 'Inter' (sans-serif) for a modern, clean, and readable design.
- Use professional, minimalist icons for categories and interactions.
- Grid-based layout to organize videos and content effectively.
- Subtle animations on hover to improve interactivity and engagement.
